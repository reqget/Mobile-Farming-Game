# Mobile Farming Game

## Overview

Mobile Farming Game is a simple farming simulation game developed with Unity.
The player can interact with the environment, grow crops, and manage farming activities.

---

## Features

* Character movement
* Planting and harvesting system
* Crop growth mechanics
* Basic inventory system

---

## Technologies

* Unity
* C#

---

## Gameplay

The player can plant seeds, water crops, and harvest them when they are ready.
